#Fileorganizer

## Filelengthfinder
This is a little program to search files with a filename which has a minumum length.
You can decide if you want to inculude foldernames or not and if you want to use the full path as name.

## Usage
With
```bash
java -jar Filenamelengthfinder.jar
```
it askes you with which arguments to run.

With
```bash
java -jar Filenamelengthfinder.jar OPTIONS
```
you can dirictly set the arguments.
To see all available options use:
```bash
java -jar Filenamelengthfinder.jar --help
```
